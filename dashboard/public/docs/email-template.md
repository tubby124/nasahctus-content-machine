# Email Templates — Content Distribution
**Purpose:** Voice-matched email templates for every content type. Connects social → email list → client relationship.
**Voice rule:** Read BRAND_VOICE.md first. Casual but sharp. No corporate filler. Subject lines are hooks, not announcements.

---

## HOW TO USE THIS FILE

Every cluster produces ONE email (sent after the video cluster drops).
Email goes to your list and/or specific prospects who would care about that topic.
Format: short intro → 2-3 punchy lines → "watch it here" link → one question to reply to.

**Email anatomy:**
1. Subject line (hook — curiosity gap or specific number)
2. One-sentence opener (personal, not promotional)
3. The insight in 3-5 sentences max
4. Link + CTA
5. Open question (drives replies → warms the list)
6. Sign-off (always the same — Hasan-flavored)

## Email Norms (always apply)

| Rule | Why |
|---|---|
| Subject line max 8 words | Mobile preview cuts off at 9 |
| No "I hope this email finds you" | Instant credibility killer |
| First line ≠ restate the subject | Wastes the preview pane |
| One link only (or two max) | Multiple links = decision fatigue |
| End with a question OR a single CTA, not both | Forces one action |
| Never "Let me know if you have questions" | Passive. Ask a specific question instead |
| CC Misha only if client-related (not list emails) | Per CLAUDE.md rules |
